# Bigbank Homework – Dragons of Mugloar Bot

This program is an automated client for the **Dragons of Mugloar** scripting adventure game – https://dragonsofmugloar.com/.

It interacts with the game's REST API to start a game, analyze quests/messages, select optimal actions, and purchase items to improve the dragon's chances of success.

## Technologies

- **Java** 25
- **Maven** 3.9.12

## Building & Testing

Build and run tests:

    mvn test

## Running the Bot

    mvn exec:java
